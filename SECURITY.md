# Security Policy

We take the security of the Aether SDKs seriously and appreciate responsible disclosure.

## Reporting a vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, use one of:

- **GitHub private vulnerability reporting** — open the repository's **Security** tab and
  click **"Report a vulnerability"**.
- **Email** — [security@aetherdb.ai](mailto:security@aetherdb.ai) with a description of the
  issue, steps to reproduce (ideally a minimal proof of concept), and the affected SDK
  version.

We aim to acknowledge a report within **3 business days** and to share a remediation plan
after triage. Please allow us a reasonable window to ship a fix before public disclosure.

## Supported versions

The Aether SDKs are pre-1.0. Security fixes are published against the latest released
version of the affected package.
